# Player-Only Quick-Start Guide (What Buttons to Press First)

This guide is for a brand-new player who wants to launch the game and complete a first playable day with minimal setup confusion.

---

## 0) Start the game

1. Open the game.
2. Wait for the Splash screen.
3. Press **Skip** (or wait for auto-advance) to reach the Main Menu.

---

## 1) Main Menu: start a new run

1. On **Main Menu**, click **New Game**.
2. You will enter **World Creator** first.

---

## 2) World Creator: pick a fast baseline

Use the tab flow in this order:

1. **Appearance & Environment**
   - Leave defaults for first run.
2. **Ecology & Inhabitants**
   - Leave defaults for first run.
3. **Government & Laws**
   - Keep default laws unless you want harder/easier crime pressure.
4. **Starting Origins**
   - Pick any origin; default is fine.
5. **Survival Mechanics**
   - Keep defaults for first run.
6. Press **Continue**.

Goal: get into gameplay quickly; do not over-optimize the first world.

---

## 3) Character Creator + Household Maker: minimum setup

1. In Character Creator:
   - Set display name.
   - Choose core appearance quickly.
   - Keep remaining settings at default for your first day.
2. Continue to Household Maker.
3. In Household Maker:
   - Confirm at least one active household member.
   - Set family/home basics.
   - Press **Start Game**.

---

## 4) First 60 seconds in gameplay

When gameplay loads, do this immediately:

1. Read top HUD values:
   - needs
   - money
   - clock/time
2. Open the journal/feed panel to see latest events.
3. Use map/location navigation once so you know where interaction options appear.

If any need is near critical, solve that first.

---

## 5) First-day action sequence (safe default)

Follow this exact sequence for a stable first day:

1. **Hygiene action** (bathroom/shower).
2. **Eat/drink** (use home food if available; buy if not).
3. **Social touchpoint** (message/contact someone).
4. **Work block** (go to work or intentionally skip if testing consequence flow).
5. **Return home**.
6. Resolve one **household pressure** (cleanup/chore/repair-like action).
7. Trigger one **social event** (conversation/interaction).
8. Press **End Day** when major needs are stabilized.

---

## 6) Save, reload, verify

Before quitting:

1. Open save flow.
2. Save to a slot.
3. Return to menu or load screen.
4. Load same slot.
5. Confirm:
   - character state looks correct
   - money/time/day status are correct
   - household state appears consistent

If parity looks wrong, note the exact step and slot used.

---

## 7) “What should I click right now?” triage rules

If unsure at any moment:

1. Fix the most critical need first.
2. If money is low, prioritize income/work-safe actions.
3. If health/status is bad, prioritize treatment/rest.
4. If household tension is rising, do one stabilizing home/social action.
5. End day only after immediate collapse risks are handled.

---

## 8) Common first-run mistakes

- Spending too long in setup screens before learning gameplay loop.
- Ignoring low hydration/food until penalties chain.
- Skipping save/load check in first session.
- Triggering too many risky actions before stabilizing baseline needs.

---

## 9) Recommended first-run success checklist

- [ ] Started new game and reached gameplay scene.
- [ ] Completed hygiene + food + social touchpoint.
- [ ] Completed a work-or-skip decision and observed consequence.
- [ ] Resolved one household pressure.
- [ ] Ended day successfully.
- [ ] Saved and reloaded the same slot with expected parity.

If all six are true, your first playable loop is complete.
