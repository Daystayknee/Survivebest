# Animals and Species Reference

This document combines the hard-coded playable species, household pet hooks, and cataloged animal supply/content entries into one designer reference.

## Playable / systemic species
- **Human** — Baseline civilian species with normal aging, full access to ordinary careers, and no supernatural restrictions.
- **Vampire** — Night-active supernatural species with blood-feeding, slow aging after maturity, sunlight shelter pressure, and vampire-only skill tree branches.

## Household pet-ready species
- Registered household pets store an open string species field, so pet support is data-driven rather than enum-locked.
- Seeded pet catalog entries: Dog, Cat, Rabbit, Parrot, Turtle.
