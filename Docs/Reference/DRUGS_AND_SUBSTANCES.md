# Drugs and Substances Reference

This document combines the substance law enum with the authored runtime profiles in `SubstanceSystem` so design can review effect pacing, dependence pressure, and default legal severity in one place.

## Substance roster (16 profiles)
- **Caffeine** (`Caffeine`) — category `Stimulant`; risk `Everyday`; default law severity `Legal`; onset `0.08` hrs; duration `4` hrs; tolerance `0.02`; addiction `0.02`; withdrawal `0.08`. Summary: Coffee, tea, energy drinks, and pills that sharpen focus fast. Buff: Energy + focus. Debuff: Jitters + sleep debt. Crash: Caffeine crash lowers mood and stamina.. Rehab: Hydration coaching or sleep clinic.
