# Human Ages and Face Types Reference

This document covers life-stage progression plus portrait-shaping enums used by character creation, genetics, and portrait rendering.

## Life stages
- **Baby** — Highest dependency stage; mostly care-driven simulation and not eligible for active social/skill-heavy play.
- **Infant** — Still fully dependent, with early routine and care loops dominating the schedule.
- **Toddler** — Mobility and household disruption start rising; still heavily supervised.
- **Child** — General childhood stage for school-age dependency, learning, and family routines.
- **Preteen** — Bridge stage before teenage autonomy; useful for identity setup and early social friction.
- **Teen** — First broadly active stage for flirting, party access, fight training, and vampire compel eligibility.
- **YoungAdult** — Default active-life stage and the point where vampires begin slow-aging behavior.
- **Adult** — Stable middle-life stage for jobs, households, parenting, and long-run progression.
- **OlderAdult** — Late-career / late-family stage with continued agency but more legacy framing.
- **Elder** — Final age band, heavily legacy- and mortality-oriented.
